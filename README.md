# Bot-de-discord-con-implementacion-de-una-ia
es una bot de discord que usa una ia que identifica imagenes y busca sus diferencias. con este bot, se busca poder subir una imagen y que este trate de acertar tu colorimetria, ademas te da recomendaciones de gente y tips.

No te olvides de crear una carpeta env y ademas tener todo lo usado actualizado. puede pasar que haya errores por tener diferentes versiones.
